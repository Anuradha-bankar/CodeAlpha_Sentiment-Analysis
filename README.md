# CodeAlpha_Sentiment-Analysis

## Overview

This project analyzes Amazon customer reviews using NLP and visualizes insights in Power BI.

## Tools

* Python (NLTK, TextBlob, VADER) 
* Power BI 
* GitHub

## Features

* Sentiment Analysis (Positive, Negative, Neutral)
* Rating vs Sentiment comparison
* Sentiments over time

## Dataset 

* Amazon_reviews dataset

## Files

* Dataset (CSV)
* Power BI Dashboard (.pbix)
* Python Code
* Screenshots

## Dashboard

* powerbi dashboard of amazon review.png

## Insight

* Total ~21K+ reviews were analyzed.
* Majority of reviews are positive (~48%).
* Negative reviews are also quite high (~44%), indicating scope for improvement.
* Most users gave 1-star ratings, showing product dissatisfaction.

## Conclusion

* This project demonstrates how combining NLP techniques with data visualization can help businesses better understand customer feedback and improve decision-making.
